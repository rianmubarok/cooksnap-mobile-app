# Project Overview

## Nama Project

CookSnap — Smart Recipe from Your Kitchen

---

## Deskripsi

Aplikasi mobile berbasis AI untuk mendeteksi bahan makanan dan memberikan rekomendasi resep berdasarkan bahan yang tersedia.

---

## Main Workflow

Splash Screen
→ Onboarding
→ Login/Register
→ Home
→ Camera Scanner
→ Scan Result
→ Recipe Recommendation
→ Recipe Detail
→ Favorite/Profile

---

## Recommendation System Logic

Sistem recommendation bekerja dengan mencocokkan ingredient hasil scan dengan ingredient pada database recipe.

Recipe dengan ingredient paling cocok akan ditampilkan lebih dahulu.
