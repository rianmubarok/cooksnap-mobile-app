Bulk Import JSON

Format wajib: Array of Objects [{}, {}].

Recipes (wajib): recipe_name, description, ingredients (array), steps (array), cooking_time (angka), difficulty (Mudah|Sedang|Sulit), opsional: image_url, tags (array), source_url, video_url.

Ingredients (wajib): name, category (contoh: Sayuran, Bumbu, Karbohidrat).

Contoh penulisan siap pakai

Contoh `recipes` 

[
  {
    "recipe_name": "Nasi Goreng Kampung",
    "description": "Nasi goreng rumahan sederhana.",
    "image_url": "https://example.com/nasi-goreng.jpg",
    "ingredients": ["nasi putih", "telur ayam", "bawang putih"],
    "steps": ["Tumis bumbu", "Masukkan nasi", "Aduk hingga matang"],
    "cooking_time": 20,
    "difficulty": "Mudah",
    "tags": ["goreng", "praktis", "indonesia"],
    "source_url": "https://example.com/resep-nasi-goreng",
    "video_url": "https://example.com/video-nasi-goreng"
  }
]

Contoh `ingredients` 

[

    "name": "Tomat",
    "category": "Sayuran"
  },
