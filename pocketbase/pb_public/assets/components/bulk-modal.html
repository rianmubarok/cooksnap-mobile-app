Bulk Import JSON

Format wajib: Array of Objects [{}, {}].

Recipes (wajib): recipe_name, description, ingredients (array objek), steps (array string), cooking_time (angka menit), difficulty (Mudah|Sedang|Sulit). Opsional: image_url, tags (array string), source_url, video_url.

Format ingredients: setiap item adalah objek dengan name (wajib), quantity (angka, opsional), unit (wajib). Gunakan "unit": "secukupnya" tanpa quantity untuk bahan tanpa takaran pasti.

Ingredients (wajib): name, category — nilai valid: Sumber Protein, Sayuran, Bumbu, Bumbu Dasar, Karbohidrat, Susu & Olahan Susu, Buah, Tepung, Lainnya.

Contoh penulisan siap pakai
