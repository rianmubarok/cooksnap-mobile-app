Edit Massal JSON

Edit data secara langsung

Edit JSON di bawah ini. Anda dapat memindahkan bahan ke kategori lain dengan memindahkannya antar array, atau menambah nama bahan baru.
Perhatian: Jika Anda mengubah nama bahan di sini, sistem akan menganggapnya sebagai bahan BARU. Untuk mengubah nama (rename), harap gunakan tombol edit (pensil) di daftar bahan. Menghapus baris JSON tidak akan menghapus data.
